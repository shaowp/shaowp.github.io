    安装git

    git在win上和linux上安装稍微有点区别，主要是需要在windows下装git bash，而linux下则比较简单

    windows下安装

    首先下载git安装包,接着一路安装即可。
    完成安装之后，就可以使用命令行的 git 工具（已经自带了 ssh 客户端）了，另外还有一个图形界面的 Git 项目管理工具。

    至此，windows下git安装完毕

    Linux下安装

    如果要在 Linux 上安装预编译好的 Git 二进制安装包，可以直接用系统提供的包管理工具。在 Fedora 上用 yum 安装：
    $ yum install git-core
    在 Ubuntu 这类 Debian 体系的系统上，可以用 apt-get 安装：
    $ apt-get install git

    git的初级使用

    使用git status可以查看到当前的文件状态,Untracked files表示未被git追踪的文件

    接着，我们将newfile.txt添加到git的暂存区（staged）

    再来查看下当前状态

    我们先暂时不提交，因为只是把一个新文件加入到暂存区，现在我们要对这个文件修改内容

    我们再来一次状态显示

    从以上可以知道一个git的重要知识点，git add filename是将文件加入到到暂存区，此时如果你再去修改这个文件，那么暂存区里是没有你的修改的快照。接上面，我们修改了文件，我们来提交第一次

    可以看到我们还是有修改但是没有被加入到暂存区的文件。因此，git commit只是将暂存区的内容提交，但是没有被add到暂存区的文件和修改则不会提交


    梳理一下
    遇到一个没有被git过的文件夹，需要先git init

    将所有文件添加到暂存区 git add .
    第一次提交 git commit -m "第一次提交"
    然后就是不断的修改文件，不断的add，不断的commit
